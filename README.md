# 🌱 Agro Sustentável e Tecnologia no Campo

## 🎯 Objetivo do Projeto

Demonstrar como a tecnologia pode ser aplicada no campo (através da agricultura sustentável), promovendo práticas conscientes e o uso de energias renováveis para melhorar a produção agrícola e reduzir impactos ambientais.

---

## 🌿 Funcionalidades do Site

✔ Página inicial com conteúdo educativo sobre agricultura sustentável  
✔ Cards informativos com imagens explicativas  
✔ Quiz interativo sobre energia renovável com JavaScript  
✔ Seção sobre energias renováveis e ideias inovadoras para o campo  

---

## 💡 Ideia Inovadora

Uso de **estufas inteligentes com energia solar**, onde painéis solares geram energia para alimentar sistemas automáticos de irrigação, reduzindo o consumo de água e energia elétrica no campo.

---

## 🧩 Recursos Técnicos

### 🌐 Interatividade (JavaScript)
- Navegação entre seções sem recarregar a página  
- Sistema de quiz com feedback automático  
- Controle de exibição de conteúdo dinâmico  

---

### 🎨 UI / UX Design
- Layout responsivo para dispositivos móveis e desktop  
- Cards com imagens e efeito hover  
- Fundo temático relacionado à agricultura  
- Interface simples e intuitiva  

---

### 📱 Responsividade
- Adaptação para telas menores  
- Ajuste automático de cards e imagens  
- Texto centralizado em dispositivos móveis  

---

## 🛠️ Tecnologias Utilizadas

### 💻 Estrutura
- HTML5 (estrutura semântica do site)

### 🎨 Estilo
- CSS3 (design, cores e responsividade)

### ⚙️ Funcionalidade
- JavaScript (interatividade e lógica do quiz)

---

## 🌍 Conclusão

O projeto busca conscientizar sobre a importância da agricultura sustentável, incentivando o uso de tecnologias limpas e práticas responsáveis no campo.
